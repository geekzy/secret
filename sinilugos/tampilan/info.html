Applikasi berbasis webbase ini bernama "Sinil UGOS" yang kepanjangannya adalah Sistem Informasi Nilai Online Unpam GO Open Source. Dimana applikasi ini sengaja dibuat untuk Universitas-Universitas/Perguruan Tinggi agar para mahasiswa bisa melihat nilai-nilai mereka secara online, agar mahasiswa tidak perlu datang ke bagian TU untuk dapat melihat nilai mereka, cukup dengan online dan login dengan account mereka saja di applikasi ini maka akan terlihat nilainya. Selain pada pemanfaatan sesungguhnya diatas applikasi ini juga mempunyai manfaat lainnya dimana sebagai bahan untuk kawan-kawan kembangkan untuk dijadikan applikasi lain atau bahan skripsi. Code pada applikasi ini bebas di pergunakan, dirubah ataupun disebarluaskan karena kami buat tanpa berlisensi alias (Open Source), kami juga turut mengundang para kawan-kawan untuk ikut gabung mengembangkan applikasi ini.
